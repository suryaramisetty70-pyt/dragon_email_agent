================================================================================
📋 WHAT IS THE TERMINAL?
================================================================================

The terminal is a black screen where you type commands. It's like asking
your computer to do something by typing instead of clicking.

Windows: Press Windows key + R, type "cmd", press Enter
Mac: Press Command + Space, type "terminal", press Enter
Linux: Press Ctrl + Alt + T

================================================================================
📋 STEP 1: Open the Terminal
================================================================================

👉 DO THIS:
1. On your keyboard, find the Windows key (bottom left, has Windows logo)
2. Press it and hold
3. While holding, press the R key
4. A small window opens
5. Type exactly: cmd
6. Press Enter

👉 NOW YOU SEE: A black screen with white or green text
   It will look like this:
   
   C:\Users\YourName>_
   
================================================================================
📋 STEP 2: Go to the Project Folder
================================================================================

👉 DO THIS:
Type this exactly and press Enter:

    cd ~/dragon_email_agent

👉 WHAT YOU SEE: The text should change to show you're in the folder
   
   C:\Users\YourName\dragon_email_agent>_

================================================================================
📋 STEP 3: Install Required Programs (Do only once!)
================================================================================

👉 DO THIS:
Type this exactly and press Enter:

    pip install loguru pydantic pydantic-settings sqlalchemy fastapi uvicorn python-dateutil beautifulsoup4 apscheduler pyyaml

👉 WHAT HAPPENS:
- It will show lots of text scrolling
- Wait until it says "Successfully installed..." or goes back to the prompt
- This takes 1-2 minutes

================================================================================
📋 STEP 4: Run the Email Agent!
================================================================================

👉 DO THIS:
Type this exactly and press Enter:

    python3 main.py

👉 WHAT YOU SEE:
- Lots of "INFO" messages scrolling
- Then it will say: "Dragon Email Agent is running. Type 'help' for commands."
- Then it will say: "Dragon> "

================================================================================
📋 STEP 5: See Your Emails!
================================================================================

Now you're inside the program. The screen says "Dragon> "

👉 DO THIS - Type each command one by one:

1. To see all commands, type:
   help
   Then press Enter

2. To sync/check emails, type:
   sync
   Then press Enter

3. To see inbox status, type:
   status
   Then press Enter

4. To see unread emails, type:
   @dragon show unread
   Then press Enter

5. To quit, type:
   exit
   Then press Enter

================================================================================
📋 WHAT TO DO IF SOMETHING GOES WRONG
================================================================================

❌ ERROR: "pip is not recognized"
------------------------------
Fix: You need to install Python properly
1. Go to https://python.org/downloads
2. Download Python
3. During install, CHECK "Add Python to PATH"
4. Restart terminal

❌ ERROR: "python3 is not recognized"
---------------------------------
Fix: Type "python" instead of "python3"

❌ ERROR: "No module named..."
----------------------------
Fix: Run the pip install command again

❌ ERROR: "Permission denied"
--------------------------
Fix: Add "sudo" before the command
   Example: sudo pip install ...
